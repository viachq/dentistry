Тема дипломної роботи:
Розроблення web-орієнтованої інформаційної системи управління стоматологічною клінікою

Diploma topic:
Development of a web-oriented information management system for a dental clinic

────────────────────────────────────────────────────────────
ОПИС ПРОЕКТУ
────────────────────────────────────────────────────────────

Мета роботи: розробити повнофункціональну web-систему для автоматизації роботи
стоматологічної клініки, що охоплює запис пацієнтів, ведення медичних карток,
облік фінансів та управління персоналом.

Об'єкт дослідження: процеси управління стоматологічною клінікою (запис на прийом,
медична документація, фінансова звітність, клієнтська база).

Предмет дослідження: методи та засоби розроблення web-орієнтованих інформаційних
систем на основі сучасних фреймворків та технологій.

────────────────────────────────────────────────────────────
ФУНКЦІОНАЛЬНІ МОЖЛИВОСТІ
────────────────────────────────────────────────────────────

Для пацієнта:
  • Реєстрація та авторизація в системі
  • Перегляд профілів лікарів та переліку послуг
  • Онлайн-запис на прийом з вибором лікаря, послуги та зручного часу
  • Перегляд і скасування своїх записів
  • Застосування промокодів на знижку
  • Написання відгуків про лікарів після відвідування
  • Перегляд сповіщень про зміни статусу запису

Для лікаря:
  • Особистий кабінет з календарем записів
  • Перегляд списку своїх пацієнтів та їхньої кількості візитів
  • Ведення приватних нотаток про пацієнтів
  • Перегляд власних відгуків та доходів
  • Управління своїм профілем та фото

Для адміністратора:
  • Управління лікарями (додавання, редагування, видалення, фото)
  • Управління послугами та їх цінами
  • Управління розкладом лікарів та винятками (вихідні, відпустки)
  • Перегляд усіх записів та зміна їхнього статусу
  • Фінансова звітність: виручка, витрати, виплати лікарям
  • Управління промокодами
  • Модерація відгуків пацієнтів
  • Перегляд бази пацієнтів
  • Налаштування інформації про клініку

Медична картка пацієнта:
  • Група крові, алергії, хронічні хвороби
  • Журнал стоматологічних візитів (діагноз, лікування, рецепти,
    рекомендації до наступного візиту)

────────────────────────────────────────────────────────────
ТЕХНОЛОГІЧНИЙ СТЕК
────────────────────────────────────────────────────────────

Backend:
  • Python 3.12, FastAPI, SQLAlchemy 2.0, Alembic
  • PostgreSQL 16, psycopg3
  • JWT-автентифікація, хешування паролів Argon2
  • Рольова модель доступу: PATIENT / DOCTOR / ADMIN

Frontend (клієнтська частина):
  • React 19, TypeScript, Vite, Tailwind CSS
  • Axios, React Router

Admin-frontend (кабінет персоналу):
  • React 19, TypeScript, Vite, Tailwind CSS
  • Axios, React Router

Інфраструктура:
  • Docker Compose (база даних)
  • REST API з документацією Swagger / OpenAPI
